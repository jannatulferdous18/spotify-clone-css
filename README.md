# 🎵 Spotify Clone (CSS)

A modern Spotify-inspired user interface clone built using **HTML and CSS**.  
This project focuses on recreating the visual experience of Spotify with a clean layout, dark theme, responsive design, and reusable styling techniques.

This project was created as a frontend development practice project to improve my skills in **HTML structure, CSS styling, layouts, positioning, and responsive design**.

---

## 🚀 Live Demo

https://jannatulferdous18.github.io/spotify-clone-css/

## ✨ Features

- 🎧 Spotify-inspired dark user interface
- 📱 Responsive layout for different screen sizes
- 🎨 Modern card-based design
- 🖼️ Album and playlist sections
- 🎵 Music player UI design
- 📌 Sidebar navigation
- ⚡ Smooth hover effects and transitions
- 🧩 Clean and organized CSS structure

## 🛠️ Technologies Used

- **HTML5**  
  Used for creating the structure and layout of the application.

- **CSS3**  
  Used for styling, responsiveness, animations, flexbox, and grid layouts.
